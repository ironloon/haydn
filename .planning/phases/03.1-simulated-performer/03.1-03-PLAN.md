1. Create `crates/haydn-performer/src/synth/waveform.rs` — Layer 2 waveform generators:
   - `fn saw_sample(phase: f32) -> f32`: Bandlimited sawtooth (additive synthesis, sum first 20 harmonics to avoid aliasing: `sum(k=1..20) (-1)^(k+1) * sin(2π * k * phase) / k`)
   - `fn triangle_sample(phase: f32) -> f32`: Bandlimited triangle (sum odd harmonics: `sum(k=0..10) (-1)^k * sin(2π * (2k+1) * phase) / (2k+1)^2`)
   - `fn square_sample(phase: f32) -> f32`: Bandlimited square (sum odd harmonics: `sum(k=0..10) sin(2π * (2k+1) * phase) / (2k+1)`)
   - `WaveformSource` struct: like SineSource but uses blended waveform instead of pure sine
   - Blend: 50% saw + 30% sine + 20% triangle (warm, rich, piano-like)
   - Implements `Iterator<Item = f32>` and `rodio::Source`
   - Add unit test: WaveformSource produces samples with higher harmonic content than SineSource (verify non-zero energy at higher frequencies)

2. Create `crates/haydn-performer/src/synth/expressive.rs` — Layer 3 expressive features:
   - `struct Lfo { rate_hz: f32, depth: f32, sample_rate: u32, sample_index: u64 }`
   - `Lfo::sample(&mut self) -> f32`: `self.depth * (2π * self.rate_hz * self.sample_index as f32 / self.sample_rate as f32).sin()`
   - `struct ExpressiveSource`: wraps a base waveform source and applies:
     - Vibrato: frequency modulation via LFO (rate=5.5Hz, depth=0.005 — subtle pitch wobble)
     - Tremolo: amplitude modulation via LFO (rate=4.0Hz, depth=0.1)
     - Velocity: scale amplitude by `velocity as f32 / 127.0`
   - Implements `Iterator<Item = f32>` and `rodio::Source`
   - Add unit test: ExpressiveSource with vibrato produces varying frequency (compare sample sequences)

3. Update `crates/haydn-performer/src/synth/builtin.rs`:
   - Import waveform and expressive modules
   - Fidelity dispatch:
     - 0: SineSource (existing)
     - 1: SineSource + ADSR (existing)
     - 2: WaveformSource + ADSR (new)
     - 3: ExpressiveSource wrapping WaveformSource + ADSR (new)
   - Higher fidelity values clamp to 3

4. Update `crates/haydn-performer/src/synth/mod.rs`: Add `pub mod waveform;` and `pub mod expressive;`
