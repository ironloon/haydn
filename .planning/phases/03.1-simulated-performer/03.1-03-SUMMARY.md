# Plan 03.1-03 Summary: Rich Synthesis + SoundFont

## What Was Done

### Task 1: Rich Waveform Synthesis (Layers 2-3)
- Created `synth/waveform.rs` with bandlimited saw, triangle, and square generators
  - Additive synthesis (20 harmonics for saw, 10 for triangle/square)
  - WaveformSource blends 50% saw + 30% sine + 20% triangle
- Created `synth/expressive.rs` with LFO-based vibrato (5.5Hz) and tremolo (4.0Hz)
  - ExpressiveSource wraps waveform generation with pitch/amplitude modulation
  - Velocity dynamics scale amplitude proportionally
- Updated `builtin.rs` to dispatch fidelity 0-3 with clamping

### Task 2: SoundFont Backend (Layer 4)
- Added `rustysynth = "1.3"` dependency
- Created `synth/soundfont.rs` implementing SynthBackend
  - Loads .sf2 files, creates rustysynth Synthesizer
  - Pre-renders note sequences with note_on/note_off + 50ms release tail
  - Stereo interleaved output via SamplesBuffer
  - Graceful error handling with download guidance for missing SF2 files
- Wired CLI `--synth soundfont --soundfont path.sf2` in main.rs
