    **Step 2: Create crates/haydn-tuning/src/engine.rs**

    Define internal compiled types:
    ```rust
    pub struct TuningEngine {
        rules: Vec<CompiledRule>,
        root_note: u8,
        metadata: Metadata,
        last_note: Option<u8>,
    }

    struct CompiledRule {
        trigger: Trigger,
        action: Action,
    }

    enum Trigger {
        ExactNote(u8),
        NoteRange { from: u8, to: u8 },
        Interval(i8),
    }

    enum Action {
        Push(i64),
        PushOffset { root_note: u8 },
        Op(haydn_vm::Opcode),
    }
    ```

    Implement `TuningEngine::compile(raw: RawTuningFile) -> Result<TuningEngine, Vec<TuningError>>`:
    1. Call `validate(&raw)` — if errors non-empty, return Err with all errors
    2. Build rules vec in evaluation order (per D-05: shorthands first, then raw):
       a. If `values` section exists → add one `NoteRange { from: range[0], to: range[1] }` rule with `PushOffset { root_note }`
       b. If `operations` table exists → for each entry sorted by note number ascending, parse key as u8, parse opcode string, add `ExactNote(key)` rule with `Op(opcode)`
       c. For each raw rule → convert trigger + emit to compiled form
    3. Store `root_note` from metadata
    4. Return `TuningEngine { rules, root_note, metadata, last_note: None }`

    Implement `pub fn map_note(&mut self, midi_note: u8) -> Option<haydn_vm::Event>`:
    1. Linear scan through `self.rules`:
       - `ExactNote(n)` → match if `midi_note == n`
       - `NoteRange { from, to }` → match if `from <= midi_note && midi_note <= to`
       - `Interval(semitones)` → match if `last_note` is Some and `(midi_note as i16 - last_note as i16) == semitones as i16`
    2. On first match, compute the Event:
       - `Push(v)` → `Event::Push(v)`
       - `PushOffset { root_note }` → `Event::Push(midi_note as i64 - root_note as i64)`
       - `Op(opcode)` → `Event::Op(opcode)`
    3. Update `self.last_note = Some(midi_note)` (ALWAYS, even on no match — per research §9, last_note persists)
    4. Return the matched Event or None

    Implement `pub fn metadata(&self) -> &Metadata`
    Implement `pub fn coverage_warnings(&self) -> Vec<String>` — delegate to validate logic (check which opcodes are mapped)
    Implement `pub fn reset(&mut self)` — set `last_note = None`

    **Step 3: Add public API functions to lib.rs**

    ```rust
    pub mod engine;

    const DEFAULT_PIANO_TOML: &str = include_str!("../tunings/piano.toml");

    pub fn load_tuning(toml_content: &str) -> Result<TuningEngine, Vec<TuningError>> {
        let raw: RawTuningFile = toml::from_str(toml_content)
            .map_err(|e| vec![TuningError::Parse(e)])?;
        TuningEngine::compile(raw)
    }

    pub fn load_tuning_file(path: &std::path::Path) -> Result<TuningEngine, Vec<TuningError>> {
        let content = std::fs::read_to_string(path)
            .map_err(|e| vec![TuningError::Io(e)])?;
        load_tuning(&content)
    }

    pub fn default_piano_tuning() -> TuningEngine {
        load_tuning(DEFAULT_PIANO_TOML)
            .expect("built-in piano tuning must be valid")
    }
    ```
